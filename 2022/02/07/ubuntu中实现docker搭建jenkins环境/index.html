准备环境

  1. Ubuntu系统（Ubuntu 18.04 LTS）
  2. Docker（v.20.10.12）

新创建的云服务器 Ubuntu 系统远程登录 ssh 默认是关闭的，需要在 /etc/ssh/sshd_config 文件中增加 PermitRootLogin yes
